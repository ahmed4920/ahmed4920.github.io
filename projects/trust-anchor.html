Vulnerability Management Automation with Nuclei

Trust Anchor Capstone Project | Cybersecurity

A senior capstone project focused on improving vulnerability management through automated security scanning and custom vulnerability detection templates.

Project Overview

Partnered with Trust Anchor to develop and improve vulnerability management processes by creating custom Nuclei templates to identify web application security misconfigurations.

Problem

Traditional vulnerability scanners may not detect organization specific security issues. The goal was to improve detection capabilities through custom automated scanning rules.

Solution

Developed custom YAML based Nuclei templates that automated the identification of common web application security issues. Testing was performed using AWS EC2 infrastructure.

Technologies Used

Nuclei YAML AWS EC2 Linux Git GitHub Web Security

Key Contributions

  • Created custom Nuclei vulnerability detection templates
  • Automated vulnerability scanning workflows
  • Developed templates for security misconfigurations including:
  • Missing Security Headers
  • OpenAPI / Swagger Exposure
  • CORS Wildcard Credential Issues
  • Open Redirect Vulnerabilities
  • Executed scans from AWS EC2 environment
  • Analyzed findings and presented results to stakeholders
